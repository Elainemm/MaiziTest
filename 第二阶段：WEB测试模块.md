# 第二阶段：WEB测试模块
  在网站架构上，WEB系统的B/S架构由于其灵活的适配和标准化等特点，越来越受到企业架构师的亲睐，在移动互联网时代，WEB系统架构更加大受欢迎。本课程将专门针对WEB系统的软件测试技术和原理进行深入讲解，包括从WEB系统架构，网络协议到安全性测试，前端测试，可用性及兼容性测试等软件测试领域均有详细讲解，并结合实战演练和案例演示，彻底解开WEB系统的神秘面纱。
